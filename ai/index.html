← 컨트롤 센터
외부 AI 에이전트·MCP 클라이언트가 BESS AI Control Center의 82인 전문가 조직과 지식 성장 데이터를 읽는 방법입니다.

① MCP 서버 연결

Claude Code 등 MCP 클라이언트에서 다음 명령으로 Streamable HTTP 서버를 등록하세요(무인증 · 레이트 리밋만 적용):

claude mcp add --transport http bess https://bess-ai-control.work/api/mcp

등록 후 제공되는 도구 3종:

도구설명
list_expertsBESS EPC AI 전문가 82명의 명부를 조회한다(ID·부서·전문 분야). division으로 필터 가능: CTO(기술)/CFO(재무)/COO(운영).
get_expert_definition특정 전문가의 역할 정의·전문 지식 전문을 조회한다. agentId는 list_experts의 ID(예: BAT-001).
ask_expert특정 BESS 전문가에게 질문하고 Claude 기반 답변을 받는다(레이트 리밋: IP당 분당 8회).

② 기계가독 리소스

LLM·크롤러가 직접 파싱할 수 있는 정적 JSON/텍스트 리소스입니다(동일 출처, 인증 불필요).

  • llms.txt— 조직·리소스 인덱스
  • llms-full.txt— 전문가 82명 전체 정의
  • agents.json— 구조화 명부(ID·key·부서·키워드)
  • graph.json— 조직·협업 관계 그래프
  • knowledge/stats.json— 에이전트별 축적 지식·레벨 통계

③ 지식 성장 데이터

82인 전문가 조직은 브리핑·스킬 갱신·팀 토론을 통해 매일 지식을 축적합니다. 누적·최근 성장 타임라인은 /growth/에서 확인할 수 있으며, 원장(knowledge/stats.json·knowledge/recent.json)은 매일 07:30 자동 갱신됩니다.

④ 이용 정책

레이트 리밋MCP·채팅 백엔드 모두 IP당 분당 8회로 제한됩니다. 초과 시 잠시 후 다시 시도해 주세요.

실시간 상담대시보드 콘솔의 "실시간 ON" 모드는 소유자 전용 액세스 키가 있어야 동작합니다. 키가 없으면 자동으로 무API 정적 매칭 응대로 전환됩니다.
